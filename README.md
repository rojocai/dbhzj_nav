# 东白湖自家导航网站 V1.1

东白湖自家导航网站，个人书签和导航站。

## 功能

- 前端导航页面（index.html）
- 后台管理页面（admin.html）
- Node.js API 服务（server.js）
- 数据存储于 JSON 文件

## 部署

Node API 服务运行在 5001 端口，通过 Nginx Proxy Manager 反代。

## 文件结构

- `index.html` - 前台导航页面
- `admin.html` - 后台管理页面
- `server.js` - Node.js API 服务
- `config.json` - 配置文件
- `data.json` - 导航数据
- `users.json` - 用户数据
- `nav-site-dongbaihu.tar.gz` - 完整打包文件
